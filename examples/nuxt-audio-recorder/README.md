# Nuxt Audio Recorder Example

Nuxt 4 + Vue 3 example demonstrating SARAUDIO audio recording with VAD and audio metering.

## Features

- Voice Activity Detection (VAD)
- Real-time audio level metering
- Speech segment recording
- Tailwind CSS 4

## Development

```bash
pnpm serve
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- **Nuxt 4** - Vue framework
- **Vue 3** - Composition API with TypeScript
- **Tailwind CSS 4** - Styling
- **@saraudio/runtime-browser** - Audio recording
- **@saraudio/vad-energy** - Voice activity detection
- **@saraudio/meter** - Audio level metering
